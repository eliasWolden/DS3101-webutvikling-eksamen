Introduction

This page will help you get familiar with our Formula 1 API, and show you how its used.

Web API

Base url: http://localhost:5257/api

The url has information about Formula 1. THe requests are GET, POST, PUT, and DELETE requests. They go over HTTP and return JSON data.

                  
                    

                      GET http://localhost:5257/api
                    

                    

                      {
                        "Driver": "http://localhost:5257/api/driver",
                        "Race": "http://localhost:5257/api/race",
                        "Team": "http://localhost:5257/api/team",
                        "Image": "http://localhost:5257/api/Image"",
                      },

There are four types available.

  • Driver: used to get all drivers
  • Race: used to get all races.
  • Team: used to get all teams.
  • Image: used to get all images.

Driver

We have a total of 22 drivers.

Driver schema

Key Type Description
id int The id of the driver
name string The name of the driver
age int The age of the driver
nationality string The nationality of the driver
image string Driver's image name used to find the image for the specific driver
teamId int Foreign key that correlates with team schema

Get all drivers

You can access the list of drivers by using the /Driver endpoint.

                  
                    

                      GET http://localhost:5257/api/driver
                    

                    

                      {
                        "id": 1,
                        "name": "Max Verstappen",
                        "age": 26,
                        "nationality": "Netherlands",
                        "image": "max-verstappen.png",
                        "teamId": 1
                      },

Get by Id

You can get a driver by using id as a parameter: /driver/2

                  
                    

                      GET http://localhost:5257/api/driver/id/2
                    

                    

                      {
                        "id": 2,
                        "name": "Sergio Perez",
                        "age": 33,
                        "nationality": "Italy",
                        "image": "sergio-perez.png",
                        "teamId": 1
                      }

Delete by name

You can delete a driver by using their name as a parameter: /Driver/Lewis%20Hamilton

                  
                    

                      DELETE http://localhost:5257/api/Driver/Lewis%20Hamilton
                    

                    

                      {
                        "id": 3,
                        "name": "Lewis Hamilton",
                        "age": 38,
                        "nationality": "United Kingdom",
                        "image": "lewis-hamilton.png",
                        "teamId": 2
                      },

Create driver

You can create a new driver by giving it a new name, age, nationality, image, and a teamId(1-10)

                  
                    

                      POST http://localhost:5257/api/Driver/
                    

                    

                      {
                        "id": 23,
                        "name": "New driver",
                        "age": 40,
                        "nationality": "Norway",
                        "image": "new-driver.png",
                        "teamId": 10
                      }

Update

You can update a driver by using their id to get their information, then edit that information(name, age, nationality, image, and teamId(1-10)) and save it.

                  
                    

                      PUT http://localhost:5257/api/Driver/
                    

                    

                      {
                        "id": 3,
                        "name": "Lewis Hamilton",
                        "age": 90,
                        "nationality": "United Kingdom",
                        "image": "lewis-hamilton.png",
                        "teamId": 2
                      }

Race

We have a total of 20 races.

Race schema

Key Type Description
id int The id of the race
WinnerName string The name of the driver that won
WinnerTime string The time it took to win the race
GrandPrix string The name of the race
NumberOfLaps int The number of laps in the race
image string image of the race

Get all drivers

You can access the list of races by using the /Race endpoint.

                  
                    

                      GET http://localhost:5257/api/Race
                    

                    

                      {
                        "id": 1,
                        "WinnerName": "Max Verstappen",
                        "WinnerTime": "1:33:56.736",
                        "GrandPrix": "Bahrain",
                        "NumberOfLaps": 57,
                        "image": "Bahrain-Grand-Prix.png"
                      },

Get by GrandPrix

You can get a race by using grandPrix as a parameter: /Race/Australia

                  
                    

                      GET http://localhost:5257/api/Race/grandPrix/Australia
                    

                    

                      {
                        "id": 3,
                        "winnerName": "Max Verstappen",
                        "winnerTime": "2:32:38.371",
                        "grandPrix": "Australia",
                        "numberOfLaps": 58,
                        "image": "Australia-Grand-Prix.png"
                      }

Delete race by id

You can delete a race by using their id as a parameter: /Race/5

                  
                    

                      DELETE http://localhost:5257/api/Race/5
                    

                    

                      {
                        "id": 5,
                        "winnerName": "Max Verstappen",
                        "winnerTime": "1:27:38.241",
                        "grandPrix": "Miami",
                        "numberOfLaps": 57,
                        "image": "Miami-Grand-Prix.png"
                      }

Create

You can create a new race by giving it a new winnerName, winnerTime, grandPrix, numberOfLaps and an image

                  
                    

                      POST http://localhost:5257/api/Race
                    

                    

                      {
                        "id": 21,
                        "winnerName": "new winner",
                        "winnerTime": "1:20:10.901",
                        "grandPrix": "new",
                        "numberOfLaps": 40,
                        "image": "new-Grand-Prix.png"
                      }

Update

You can update a new race by using their id to get their information, then edit that information(winnerName, winnerTime, grandPrix, numberOfLaps and an image)

You can update a team by using their id to get their information, then edit that information(manufacturer, image, driver1, and a driver2) and save it.

                  
                    

                      PUT http://localhost:5257/api/Race
                    

                    

                      {
                        "id": 2,
                        "winnerName": "Sergio Perez",
                        "winnerTime": "1:21:14.894",
                        "grandPrix": "Edited race",
                        "numberOfLaps": 50,
                        "image": "Saudi-Arabia-Grand-Prix.png"
                      }

Team

We have a total of 10 teams.

Team schema

Key Type Description
id int The id of the team
manufacturer string The producer of the car
image string Image of the team
driver1 string The main driver
driver2 string The supporting driver

Get all teams

You can access the list of teams by using the /Team endpoint.

                        "driver2": "Carlos Sainz"
                      },

Get team by id
